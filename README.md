# 🚀 InetProxy

Автоматический SOCKS5 прокси-роутер на базе Xray-core. Скачивает конфиги из подписок, проверяет на работоспособность и поднимает локальные SOCKS5 прокси с логином и паролем. 

Один процесс Xray обслуживает все порты, потребляя минимум оперативной памяти.

## ⚡ Возможности
- Мультипротокол: VLESS (Reality), VMess, Shadowsocks, Trojan.
- Умная проверка: проверяет прокси батчами, не перегружая сеть.
- Авто-лечение: мертвые прокси автоматически заменяются на живые из запасов.
- Строгая нумерация: порты идут строго по порядку (10001, 10002...) без "дыр".
- Безопасность: все SOCKS5 защищены логином и паролем.

## 🛠 Быстрая установка

Скрипт автоматически установит все нужные зависимости (Go, Xray-core), скомпилирует софт и создаст шаблоны конфигураций:

```bash
git clone https://github.com/internetkafe/inetproxy.git
cd inetproxy
sudo bash install.sh
```

## ⚙️ Настройка

Перед запуском отредактируйте два файла в папке проекта:

1. **`.env`** — укажите данные для доступа к вашим прокси:
   ```env
   SOCKS_USER="ваш_логин"
   SOCKS_PASS="ваш_пароль"
   START_PORT=10001
   MAX_PROXIES=50
   RECHECK_INTERVAL=60
   ```

2. **`subs.txt`** — добавьте ссылки на подписки (каждую с новой строки):
   ```text
   https://your-provider.com/sub1
   https://another-provider.com/sub2
   ```

## 🚀 Запуск

```bash
./inetproxy
```

Программа скачает подписки, проверит прокси на живучесть и выведет готовый список рабочих `IP:PORT`. Всё, прокси готовы к использованию!
